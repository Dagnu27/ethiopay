EthioPay: A digital payment platform for Ethiopia.

 ✨ Features
- Send/receive money
- Pay bills
- QR payments
- Transaction history
- Admin dashboard

 🛠️ Stack
         React + Tailwind + Node.js + Express + PostgreSQL

 🚀 Quick Start
      bash
      cd server && npm install && npm run dev
       cd client && npm install && npm run dev
